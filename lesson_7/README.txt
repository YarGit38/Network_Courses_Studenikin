Создайте Maven проект с 2 maven сабмодулями для сервера и клиента.

Сервер должен отвечать строкой "Hello <значение name из запроса>" 
на HTTP GET запросы по адресу 127.0.0.1:8080/service?name=example.

Клиент должен выполнять запросы по адресу 127.0.0.1:8080/service?name=example 
и ответ от сервера выводить на консоль.
